# Spec: Team Admin Detail

## Goal
- Strengthen team administration with a dedicated detail screen and richer team context.

## Requirements
- Admin can open a dedicated team detail page.
- Team detail must show core team data, roster, active/pending counts, and modalities represented.
- Team detail must support update and safe delete actions.
- Team list should link to the team detail page.
- Add automated backend tests if backend response shape changes.
