# Feature Specification: ALDEBARUN Public Module

## Summary

Deliver a dedicated ALDEBARUN II public page backed by real data for race modalities, registrations CTA, results, and time rankings.

## Requirements

- The backend must expose real ALDEBARUN race modalities and rankings.
- The public frontend must provide a dedicated `/aldebarun` page.
- The page must highlight registrations, race modalities, and rankings without mock data.
- The implementation must include automated tests and technical status sync.
